🚀 Hou-Swoole 是一个简单、轻量并且拥有超高性能的PHP协程框架。

## 性能测试

基准测试性能排名领先于Java、Go、Python等语言的Web框架。

## 支持功能

* [x] HTTP Server
* [x] WebSocket Server
* [x] TCP Server
* [x] UDP Server
* [x] MQTT Server/Client
